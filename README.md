# quickstart

**HTML5** + **JavaScript (CommonJS)** + **SCSS** + LiveReload

Clone:

    $ git clone --depth=1 --branch=master git@github.com:romainl/quickstart.git dir
    $ rm -rf !$/.git

Install:

    $ npm install

Build:

    $ npm run all

or

    $ make all

Compile:

    $ npm run code

or

    $ make code

Watch:

    $ npm run watch

or

    $ watch -n 1 make code
